# 三种垂直输送方法图设计

## 目标

基于2023年4月至10月HF、CF站点现有ERA5与WRF统一高度插值数据，仿照第四节报告原图三的四面板布局，生成WRF三种垂直输送量及ERA5垂直平流通量图。只输出PNG图片，不新增CSV或Excel。

## 数据与范围

- ERA5输入：`ERA5_HF_CF_202304-202310_逐小时0-4km插值.csv`。
- WRF输入：`WRF_HF_CF_202304-202310_逐小时0-4km插值.csv`。
- 站点：HF、CF。
- 时间：北京时间08:00–16:00。
- 高度：相对地面0–4 km，每100 m一层。
- 共同期：2023-04-01至2023-10-31。
- 垂直速度符号：正值向上，负值向下。

## 四种图中量

1. WRF垂直平流通量：`F_adv = C_O3_WRF * wa_WRF`。
2. WRF湍流扩散通量：`F_turb = -Kz_WRF * dC_O3_WRF/dz`，其中`Kz_WRF`使用WRF插值后的`exch_h_source_value`。
3. WRF综合垂直输送：`F_total = F_adv_WRF + F_turb_WRF`，两项均使用WRF变量。
4. ERA5垂直平流通量：`F_adv = C_O3_ERA5 * w_geo_ERA5`。

WRF三种方法只使用WRF臭氧及WRF动力/交换变量；ERA5平流方法只使用ERA5臭氧与ERA5几何垂直速度。两套数据不交叉混用。

O3浓度单位为`ug m-3`，垂直速度单位为`m s-1`。若WRF `EXCH_H`按其Registry物理定义解释为`m2 s-1`，四种通量单位均为`ug m-2 s-1`。由于本地WRF后处理文件中`EXCH_H`单位属性为`-`，WRF湍流与综合通量图必须明确注明“按m2 s-1假定”。

## 梯度和统计

- 对每个站点、每个北京时间时次，沿100 m目标高度网格计算`dC_O3/dz`。
- 中间高度采用中心差分，边界采用单边差分，由NumPy梯度函数完成。
- 时间－高度矩阵为共同期内按站点、北京时间小时和高度的多年平均。
- 日间平均廓线为共同期内按站点和高度对08:00–16:00全部记录求平均。
- 不对有方向的通量取绝对值。

## 图形布局

每个通量生成一张四面板图：

- (a) HF时间－高度分布。
- (b) HF日间平均廓线。
- (c) CF时间－高度分布。
- (d) CF日间平均廓线。

复用`plot_figure2_shanghai_hefei_era5_tfw.py`的2×2布局、宽度比例、字号、零线和色标位置。每张图的HF/CF共用以0为中心的对称色标，显示范围采用该方法全部有限值绝对值的98百分位并作易读取整；真实数据不裁剪，只允许图面色彩饱和。

- 检查四图均包含四个面板且非空。
- 检查横轴为08–16时、纵轴为0–4 km。
- 抽查`F_adv=C*w`、`F_turb=-Kz*dC/dz`和`F_total=F_adv+F_turb`。
- 检查所有色标以0为中心，正值向上、负值向下。
- 检查WRF湍流与综合图包含`EXCH_H`单位假设说明。
