# BLH相对分层统计设计

## 目标

仅使用2023年4-10月HF/CF的ERA5和WRF统一插值数据，定量比较边界层内部、边界层顶附近和边界层以上的O3、垂直运动及输送通量差异。不读取或配对任何`SHHKY`本地廓线数据。

## 数据与范围

- ERA5：`ERA5_HF_CF_202304-202310_逐小时0-4km插值.csv`
- WRF：`WRF_HF_CF_202304-202310_逐小时0-4km插值.csv`
- 站点：HF城市站、CF农村站
- 时间：北京时间08:00-16:00
- 高度：距地0-3 km

## 动态分层

每条记录使用同一站点、同一小时的BLH/PBLH：

1. 边界层内：`height_agl_m <= pblh_m`
2. 边界层顶附近：`pblh_m < height_agl_m <= pblh_m + 500 m`
3. 边界层以上：`height_agl_m > pblh_m + 500 m`且不超过3 km

## 计算量

两种数据源均计算：

- O3浓度：`o3_mass_ug_m3`
- 几何垂直速度：`w_geometric_m_s`
- 垂直平流通量：`Fadv = C * w`
- 水平输送强度：`TFh = C * sqrt(u^2 + v^2)`

WRF额外计算：

- O3垂直梯度：同一站点、同一时刻沿高度使用`numpy.gradient`
- 湍流扩散通量：`Fturb = -EXCH_H * dC/dz`
- 总诊断通量：`Ftotal = Fadv + Fturb`

WRF的`EXCH_H`绝对单位在源文件中记录为`-`，当前绝对Fturb继续按`EXCH_H`为m2 s-1解释；图题明确标注该假设。

## 平均策略

采用两阶段平均：

1. 先对每个`source-station-bjt_time-layer`内的高度格点求层平均；
2. 再对所有小时层平均值求总体均值、标准差、中位数、四分位数和有效小时数。

这样每个小时权重相同，不会因某小时边界层较深、包含更多100 m格点而被重复加权。
