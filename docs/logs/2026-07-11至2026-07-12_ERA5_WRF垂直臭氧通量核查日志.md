# ERA5/WRF垂直臭氧通量核查日志

## 1. 基本信息

- 日期：2026-07-11至2026-07-12
- 站点：HF（合肥城市站，31.78°N、117.18°E）和CF（长丰农村站，32.21°N、117.18°E）
- 时段：2023年4月至10月，北京时间08:00-16:00
- 高度：距地高度0-4 km，100 m间隔
- 数据处理原则：ERA5使用ERA5 O3和ERA5几何垂直速度；WRF使用WRF O3、WRF几何垂直速度和WRF EXCH_H。

## 2. 已完成工作

1. 对ERA5和WRF数据分别完成0-4 km统一高度插值。
2. 生成HF、CF两站逐小时O3和几何垂直速度对比数据。
3. 按三种方法计算WRF垂直臭氧输送：
   - 垂直平流通量：`Fadv = C × w`
   - 参数化湍流扩散通量：`Fturb = -Kh × dC/dz`
   - 合计诊断通量：`Ftotal = Fadv + Fturb`
4. 使用ERA5独立计算垂直平流通量：`Fadv = C_ERA5 × w_ERA5`。
5. 生成四张HF/CF时间-高度分布及平均廓线图：WRF Fadv、WRF Fturb、WRF Ftotal和ERA5 Fadv。

正负方向统一定义为：几何垂直速度 `w > 0` 表示向上，`w < 0` 表示向下；因此逐点计算时 `Fadv > 0` 表示向上输送，`Fadv < 0` 表示向下输送。

## 3. Fadv平均顺序核查

现有代码与论文Figure 4采用相同的基本计算顺序：

1. 对每个日期、小时和高度格点逐点计算：

   `Fadv(t,z) = C(t,z) × w(t,z)`

2. 时间-高度图中的每个格点再对相同小时和高度的所有日期求平均：

   `mean_Fadv(h,z) = mean[C(t,h,z) × w(t,h,z)]`

3. 日间平均廓线同样对已经逐点得到的Fadv求平均。

因此当前结果是：

`mean(C × w)`

而不是：

`mean(C) × mean(w)`

两者关系为：

`mean(C × w) = mean(C) × mean(w) + Cov(C,w)`

现有计算自动保留了样本尺度的O3-w协方差，但没有额外人为添加“残差项”。该协方差是小时、日期或高度样本变化产生的统计协方差，不能直接等同于未解析的真实湍流通量。

代码依据：`plot_three_vertical_flux_methods_hf_cf_2023.py`第29-30行先计算逐点Fadv，第59-64行再对Fadv执行均值聚合。

## 4. 平均w与平均Fadv符号不同的实例

为解释符号差异，临时核查HF站14:00、0.2-1.8 km范围内全部日期和高度样本，得到：

- 平均O3：79.78 μg m^-3
- 平均w：+7.10×10^-5 m s^-1
- `mean(C) × mean(w)`：+0.00566 μg m^-2 s^-1
- `Cov(C,w)`：-0.01136 μg m^-2 s^-1
- 实际 `mean(C × w)`：-0.00570 μg m^-2 s^-1

其中，向上样本累计通量为+1754.99，向下样本累计通量为-1775.71。两者高度抵消，向下部分仅略强，因此最终是接近零的微弱负通量。这一实例只是对既有结果的诊断拆解，没有修改原计算方法或额外加入计算项。

## 5. 与论文Figure 4的主要差异

1. 论文O3来自同步MAX-DOAS观测廓线；当前ERA5图使用ERA5 O3，WRF图使用WRF O3。
2. 论文仅使用MAX-DOAS有效观测时刻，并与ERA5气象场逐时匹配；当前图使用2023年4-10月所有可用08:00-16:00样本。
3. 论文采用约200 m垂直网格和约0-3 km范围；当前采用100 m网格和0-4 km范围。
4. 论文根据逐层ERA5气压和温度将观测O3由ppb转换为质量浓度；当前O3质量浓度由各自模式变量转换得到。

所以两者虽然都按 `TFw = C × w` 逐点计算，但O3来源、有效时刻筛选、垂直分辨率和样本集合不同，图形结构和平均值不能直接视为完全等价。

## 6. 0-200 m输送方向诊断

2023年4-10月、北京时间08:00-16:00统计结果：

- ERA5 HF：平均w为+0.00177 m s^-1，Fadv为+0.133 μg m^-2 s^-1，平均向上。
- ERA5 CF：平均w为+0.00115 m s^-1，Fadv为+0.089 μg m^-2 s^-1，平均向上。
- WRF HF：Fadv为+0.094 μg m^-2 s^-1；Fturb为-0.178 μg m^-2 s^-1；Ftotal为-0.083 μg m^-2 s^-1，湍流扩散抵消平流后平均向下。
- WRF CF：Fadv为+0.027 μg m^-2 s^-1；Fturb为-0.015 μg m^-2 s^-1；Ftotal为+0.012 μg m^-2 s^-1，接近中性、略向上。

近地面0 m和100 m数据存在由最低有效模式高度向下补齐的情况，因此0-200 m结论应作为模式诊断，不宜解释为精确地表湍流观测。

## 7. WRF高层Fturb偏低核查

`Fturb = -Kh × dC/dz`。统计表明高层Fturb低值主要由EXCH_H在边界层顶以上急剧下降造成，而不是符号错误。

### HF站

- 0-0.8 km：平均Kh为83.35，平均|Fturb|为0.700 μg m^-2 s^-1。
- 1.9-3.0 km：平均Kh为1.09，平均|Fturb|为0.0136 μg m^-2 s^-1。
- 3.1-4.0 km：平均Kh为0.74，平均|Fturb|为0.0080 μg m^-2 s^-1。

### CF站

- 0-0.8 km：平均Kh为89.57，平均|Fturb|为0.762 μg m^-2 s^-1。
- 1.9-3.0 km：平均Kh为1.60，平均|Fturb|为0.0204 μg m^-2 s^-1。
- 3.1-4.0 km：平均Kh为0.77，平均|Fturb|为0.0082 μg m^-2 s^-1。

从低层到3-4 km，O3梯度绝对值约降低2倍，但Kh降低超过100倍，因此Fturb降低约90倍。WRF日间平均边界层高度约0.83-0.84 km，90%的边界层高度低于约1.45-1.50 km，2-4 km多数时间位于自由对流层，局地参数化湍流扩散较弱，结果具有物理合理性。

图中色标使用两站全部高度平均通量绝对值的第98百分位确定，低层大值控制色标，也会使高层视觉上更接近白色。
